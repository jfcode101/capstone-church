# Church Website

This is a MERN stack church website built with MongoDB, Express, React, and Node.js. The application allows users to register, log in, view sermons, and manage events. It aims to provide a platform for church members to engage with content and stay updated on church activities.

## Features

- User registration and login
- View and create sermons
- Manage church events
- Responsive design for mobile and desktop
- Admin dashboard for managing content (to be implemented)
- User authentication with JWT
- Error handling and input validation

## Technologies Used

- **Frontend**: React, React Router, Axios
- **Backend**: Node.js, Express, MongoDB, Mongoose, JWT, Bcrypt
- **Development Tools**: Nodemon, Postman
- **Styling**: CSS (custom styles)
